二、React

JSX

JSX需要由父标签包裹，一般使用<div>，也可以直接使用<React.Fragment>或<>（简写）。
JSX中所有标签都需要闭合。
使用{/*注释内容*/}做单行注释，使用/* 多行 */做多行注释。
在{}括号中位JavaScript表达式。
``反引号包裹模版字符串。
null也是合法的JSX元素，但不会被渲染。可以用于三元表达式。
